# 李琪 — 视觉设计师作品集网站

## 文件结构

```
求职/
├── index.html          # 主页面（单文件，含内联 CSS + JS）
├── README.md           # 本说明文件
└── portfolio/          # 作品集素材文件夹
    ├── images/         # 图片素材
    │   ├── 温压采集器.png
    │   ├── NTR300.png
    │   ├── 展会效果图1.png
    │   ├── 展会效果图2.png
    │   └── 展会效果图3.png
    ├── videos/         # 视频素材
    │   ├── 双球阀.mp4
    │   ├── CPT100.mp4
    │   └── WAC710N.mp4
    └── source/         # 可下载的源文件
        ├── 智慧供热产品手册.ai
        ├── 智慧供热解决方案手册.ai
        ├── 宣传海报1.psd
        ├── 宣传海报2.psd
        ├── 北京展会.skp
        ├── 济南展会.skp
        └── 2025北京展会.skb
```

## 使用方式

1. **本地预览**：直接用浏览器打开 `index.html` 文件即可
2. **部署上线**：将整个 `求职` 文件夹上传到任意静态托管服务（如 CloudBase 静态托管、Vercel、GitHub Pages 等）

## 功能特性

### 作品上传
- 支持拖拽或点击上传图片/视频
- 上传的文件存储在浏览器 IndexedDB 中，刷新页面不丢失
- 支持删除已上传的作品

### 作品展示
- 8 个预设作品（5 张图片 + 3 个视频）
- 6 个分类筛选按钮
- 灯箱预览（支持键盘左右切换、ESC 关闭）
- 悬停动效与信息浮层

### 设计源文件下载
- AI 手册、PSD 海报、SKP 模型等原始工程文件可直接下载

### 联系方式
- 点击手机号/邮箱/微信即可复制到剪贴板

## 技术栈

- 纯 HTML5 / CSS3 / JavaScript，无框架依赖
- IndexedDB 本地存储
- IntersectionObserver 滚动动画
- CSS Grid + Flexbox 响应式布局
- Google Fonts（Noto Sans SC + Inter）

## 注意事项

- 首次打开时 Google Fonts 可能需要联网加载
- 视频文件较大，首次加载可能需要等待
- 源文件（.ai / .psd / .skp）仅供下载，浏览器无法直接预览
